# Фильтры и маски

> Оба механизма **ломают batching** и добавляют проходы рендера — важны для собеса по производительности.

## Filters

```ts
import { Sprite, BlurFilter } from 'pixi.js';

const sprite = new Sprite(texture);
sprite.filters = [new BlurFilter({ strength: 4 })];
// или
sprite.filters = new BlurFilter({ strength: 4 }); // один фильтр без массива
```

**Как работает (упрощённо):**

1. Объект (и поддерево) рисуется во **временный RenderTexture** (pass).
2. К текстуре применяется shader фильтра.
3. Результат композится обратно на сцену.

→ Дополнительная память GPU + draw passes. Несколько фильтров — цепочка pass’ов.

### Встроенные (примеры)

- `BlurFilter`, `AlphaFilter`, `ColorMatrixFilter`, `DisplacementFilter`, `NoiseFilter`…

### padding

Фильтры расширяют bounds (`filter.padding`), иначе blur обрежется по краям bbox.

```ts
blurFilter.padding = 20;
```

## Маски

### Graphics / Sprite mask (alpha)

```ts
const mask = new Graphics().circle(0, 0, 50).fill(0xffffff);
sprite.mask = mask;
// mask должен быть в scene graph (часто addChild того же parent)
parent.addChild(sprite);
parent.addChild(mask);
```

- Маска использует **альфу** маскирующего объекта.
- `sprite.mask = mask` — обрезка по форме.

### Inverse mask

`mask.inverse` — инвертировать область (где поддерживается API версии).

## Filter vs Mask

| | Filter | Mask |
|---|--------|------|
| Эффект | Цвет, blur, искажение | Обрезка видимости |
| Стоимость | Shader pass | Доп. stencil/alpha pass |
| Batching | Ломает | Ломает |

## Оптимизация на собесе

- Меньше объектов с фильтрами одновременно.
- Не вешать blur на весь `stage`.
- Кэшировать результат: отрисовать в `RenderTexture` раз, показывать спрайтом.
- Для простой прямоугольной обрезки — **`cacheAsTexture`** / crop / viewport вместо blur-mask (зависит от задачи).

## destroy

```ts
filter.destroy();
// при destroy спрайта фильтры тоже нужно отцепить или уничтожить
sprite.filters = null;
```

## Частые вопросы

**Почему FPS упал после blur?**  
Каждый filtered object — offscreen render + full-screen quad с shader.

**Маска не работает?**  
Маска не в дереве, неверный parent, маска и объект в разных ветках без общего рендера.

**Можно ли batch спрайты с разными фильтрами?**  
Нет в одном batch — разные pipeline states.

## Шпаргалка

> Filter — post-process через RenderTexture и shader, дорого. Mask — обрезка по альфе другого объекта, тоже отдельный pass. Оба увеличивают draw calls и VRAM. Для UI-скруглений иногда дешевле nine-slice текстура, чем blur.
