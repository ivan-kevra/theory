# События и интерактивность

> В v8 единая **Federated Events** модель (как DOM): всплытие и погружение по scene graph.

## Включение интерактивности

```ts
sprite.eventMode = 'static'; // 'none' | 'passive' | 'auto' | 'static' | 'dynamic'
sprite.cursor = 'pointer';

sprite.on('pointerdown', (e) => {
  console.log(e.global.x, e.global.y);
});
```

| eventMode | Когда |
|-----------|--------|
| `none` | События не получает (по умолчанию у многих объектов) |
| `passive` | Только слушает, не участвует в hit-test как цель* |
| `static` | Кнопки, UI — hit-test кэшируется |
| `dynamic` | Движущиеся кликабельные объекты — пересчёт каждый кадр |
| `auto` | Устаревший аналог dynamic |

\*Уточняй в доке версии — на собесе важно: **без `eventMode` клики не придут**.

## Типы событий (pointer)

- `pointerdown`, `pointerup`, `pointermove`, `pointerover`, `pointerout`
- `click`, `rightclick`
- `wheel` — на scrollable UI
- Touch мапится в pointer (`pointerId`)

## Hit area

По умолчанию — прямоугольник по bounds. Для непрямоугольных кнопок:

```ts
import { Circle, Rectangle, Polygon } from 'pixi.js';

sprite.hitArea = new Circle(0, 0, 50);
// или произвольный Polygon
```

Без hitArea большой прозрачный PNG ловит клики по всему bbox — частая ошибка.

## Всплытие (bubbling)

Событие идёт от цели вверх по родителям (`e.currentTarget`, `e.target`). Можно `e.stopPropagation()`.

```ts
button.on('pointerdown', (e) => {
  e.stopPropagation(); // не отдать родителю
});
```

## Глобальные координаты

- `e.global` — позиция указателя в системе stage/renderer.
- `e.getLocalPosition(sprite)` — в локальных координатах объекта.

## EventSystem

Живёт в `renderer.events`. Один canvas — один набор слушателей DOM → Pixi переводит в federated events.

## Частые вопросы

**Клик не срабатывает?**  
`eventMode: 'none'`, объект под другим спрайтом, `visible: false`, нулевой `alpha`, родитель с `eventMode: 'none'` блокирует детей, неверный `hitArea`.

**Как drag?**  
`pointerdown` на объекте → `pointermove` на `stage`/app.stage с флагом → `pointerup`/`pointerupoutside`.

**Отличие от DOM?**  
Рендер в canvas — нет HTML-элементов на спрайтах; события синтетические через EventSystem.

**`interactive = true` (v7)?**  
В v8 заменено на **`eventMode`**.

## Шпаргалка

> Интерактивность: `eventMode: 'static'|'dynamic'`, слушатели `pointerdown` и т.д. Hit area задаёт форму клика. События всплывают по Container’ам. Координаты курсора — `e.global` и `toLocal`.
